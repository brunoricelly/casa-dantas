# Casa Dantas Quixadá

Site moderno da Casa Dantas, tradicional loja de calçados, bolsas, malas e acessórios em Quixadá-CE.

## Stack

- Astro 5
- React Islands
- TypeScript
- Tailwind CSS
- SEO + sitemap
- Docker/Nginx para deploy no Coolify

## Desenvolvimento

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deploy

Produção planejada:

- Domínio: https://casadantas.chatwoot.space
- Branch: `main`
- Coolify: Dockerfile

## Roadmap

- Integração com PostgreSQL para leads/pedidos
- Integração Garage S3 para imagens dos produtos
